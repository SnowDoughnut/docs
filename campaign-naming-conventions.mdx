<Accordion title="Concept review: campaign-level identifiers">
  These six categories describe the strategy behind a campaign, not its creative or its audience. That's what makes one campaign distinguishable from another at a glance: which channel it runs on, what it's optimizing for, and where it sits in the funnel. Not every channel uses every category. A Google Search campaign is often optimized through bid type rather than a platform-level budget type, while Meta's campaign budget optimization makes budget type the meaningful field and leaves bid type blank. When you assemble an actual name, skip whichever category a given channel doesn't use rather than inserting a placeholder into the string.

  If your program needs a seventh category, add a new table here following the same long form, short form, description pattern. Don't rename or remove one of the six above once a live campaign is using it.
</Accordion>

## Assemble the name

Not every channel uses every category. Google Search leans on bid type, so its name skips budget type:

`Google-Search-Leads-tROAS-TOFU`

Meta's campaign budget optimization makes budget type the meaningful field instead, so its name skips bid type:

`Meta-Instagram-Awareness-CBO-TOFU`

## Testing identifiers

<Tip>
  Optional: append only to sets / groups / variants that are part of an active test.
</Tip>

Testing identifiers mark a hypothesis you're currently running, for example a campaign bidding strategy test. Because a test only applies to some campaigns for a limited time, testing identifiers aren't part of the standard identifier set: append one to the end of the camapigns, after the standard identifiers, instead of building it into a table above.

- With a testing identifier appended: `Meta-Instagram-Awareness-CBO-TOFU-TEST01`

This document only covers where a testing identifier sits in the name. It doesn't cover how a test ID gets created, how long it stays on an ad, or where the result gets logged once the test ends. Visit the [Experiment ID SOP](/experiment-ids) for that.
